# cerebral-website
Cerebral website

## How to contribute

1. Fork the repo
2. Do any changes
3. Create pull request

## How to run it

1. `npm install`
2. `npm start`
3. Menu changes is in `menu.js`
4. Add markdown articles to `markdown` folder

Kinda cool here is that when you change your markdown the webpage will actually hot reload it :-)
